# About the engine
[Plus Engine](https://github.com/Psych-Plus-Team/FNF-PlusEngine) is an engine based on Psych 1.0.4 base that offers **great compatibility with Psych 0.6.3 & 0.7.3 mods**, **supports stepmania & multiple languages**, **more improved features**, and is available on several platforms such as **Windows**, **Linux**, **macOS**, **iOS**, and **Android**.

---

# Features
- Many variables for window and system management in `Lua`
- Key Viewer
- Modchart Settings
- New Gameplay Changers such as **No Drop Penalty** and **Perfect Only** (However, **Opponent Mode** is currently in maintenance)
- New accuracy systems such as **Wife3**, **ITG**, **DJMAX**, **osu!**, and **Simple**, apart from **Psych**
- Added songs with **Erect** and **Nightmare** difficulty from **V-Slice** without stages and events, only charts
- And many more!
